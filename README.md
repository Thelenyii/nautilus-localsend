# nautilus-localsend
Integration for send files with LocalSend from Nautilus
Інтеграція для надсилання файлів за допомогою LocalSend із контекстного меню у Nautilus

## Встановлення залежностей
Arch/Manjaro:
```
sudo pacman -S python-nautilus
```
## Встановлення плагіна
```
git clone https://github.com/Thelenyii/nautilus-localsend.git
cd nautilus-localsend
mv nautilus-localsend.py ~/.local/share/nautilus-python/extensions/
```
